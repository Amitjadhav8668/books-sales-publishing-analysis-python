# Books Sales & Publishing Trend Analysis using Python

## Objective
This project performs Exploratory Data Analysis (EDA) on a books dataset to identify trends in publishing year, genre distribution, pricing, ratings, and sales.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks Performed
- Data cleaning and preprocessing
- Missing value handling
- Duplicate removal
- Statistical summary analysis
- Data visualization

## Visualizations Used
- Histogram
- Bar Chart
- Boxplot
- Scatter Plot
- Pie Chart
- Line Plot

## Key Insights
- Most books in the dataset were published after the year 2000.
- Certain genres appear more frequently than others.
- Price and sales patterns provide useful business insights.
- Some authors and publishers contribute more significantly to total sales and revenue.
- Language and author rating also influence book performance.
